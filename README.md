# AI-Based Sentiment Analysis System

## 📌 Project Description
This project is a Machine Learning system that analyzes text and predicts sentiment as Positive, Negative, or Neutral.

---

## 🧠 Objective
To automatically detect human emotions from text using AI techniques.

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Logistic Regression  
- TF-IDF Vectorization  

---

## 📂 Files in This Project
- train.py → Model training  
- predict.py → Testing model  
- data.csv → Dataset  
- model.pkl → Trained model  
- vectorizer.pkl → Text processing  
- requirements.txt → Dependencies  

---

## 🚀 How to Run

### Step 1: Install libraries
pip install -r requirements.txt

### Step 2: Train model
python train.py

### Step 3: Run prediction
python predict.py

---

## 🎯 Example Output

Input: I love this product  
Output: Positive  

Input: This is bad  
Output: Negative
